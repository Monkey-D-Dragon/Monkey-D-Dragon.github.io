LMMSE DMD中的LLR计算。

LLR的计算

均衡后的解调信号模型

均衡之后得到的星座点是在标准星座点基础上缩放加平移的结果，在这种理解下可以建模为

x^=ρx+n

这里的x^表示均衡后得到的星座点，x表示标准星座点，由于标准星座点是能量归一化的，所以缩放系数ρ的取值范围只能是(0,1)，不能取大于1的值，并且这里ρ2是信号功率（不含噪声和干扰）。这里的n相当于噪声加干扰，并且服从均值为0，方差为ρ(1−ρ)的复高斯分布。关于为什么噪声的分布参数和信号功率参数（幅度）有关，暂时没有想到数学上的说明，但是可以这么理解：经过一系列公式推导后会发现，实际上ρ表示信号和噪声的总功率，ρ2表示信号的有效功率（也就是去除干扰和噪声的功率），所以干扰和噪声的功率和就是ρ−ρ2，也就是这里n的方差（这里的推导其实是SINR的推导，后面小节细说）。

接收这一信号模型之后，在其基础上进行LLR的推导。为了简化推导，需要首先明确，3gpp中星座点采用格雷码映射，实部和虚部是独立的，所以推导过程只采用实部或虚部即可。

以实部为例y=Re[x^]，另外为了说明，将信号模型中的三个部分用复数表示。

QPSK

对QPSK，信号模型的复数用x^=a+bj,x=12[(1−2b(2i))+j(1−2b(2i+1))],n=c+dj表示，可以得到实部对应y=Re[x^]=ρ(2)[1−2b(2i)]+c=ρϕ[1−2b(2i)]+c，用ϕ=12表示QPSK的归一化系数。根据复高斯分布的性质：复高斯随机变量的方差是其实部或虚部对应实高斯随机变量方差的两倍。也就是这里的n的方差是ρ(1−ρ)，所以n=c+dj的实部虚部c,d的方差都是ρ(1−ρ)2，所以c服从均值为0，方差为ρ(1−ρ)2的实高斯分布。这样就可以得到

f(c)=12πρ(1−ρ)2exp[−(y−ρϕ[1−2b(2i)])22ρ(1−ρ)2]

化简得到

f(c)=1πρ(1−ρ)exp[−(y−ρϕ[1−2b(2i)])2ρ(1−ρ)]

所以LLR为l(2i)=lnP(b(2i)=0|y)P(b(2i)=1|y)=exp[−(y−ρϕ)2ρ(1−ρ)]exp[−(y+ρϕ)2ρ(1−ρ)]=4ϕ1−ρy

如果l(2i)>0，则b(2i)=0；如果l(2i)<0，则b(2i)=1；如果不幸l(2i)=0，那么需要特殊处理。

对于虚部，只是将上面公式的b(2i)替换为b(2i+1)，其他的完全一样。

16QAM

对16QAM，信号模型的复数用x^=a+bj,x=110[(1−2b(4i))[2−(1−2b(4i+2))]+j(1−2b(4i+1))[2−(1−2b(4i+3))]],n=c+dj表示，可以得到实部对应y=Re[x^]=ρ(10)[1−2b(4i)][2−(1−2b(4i+2))]+c=ρϕ[1−2b(2i)]+c，用ϕ=110表示16QAM的归一化系数。根据复高斯分布的性质：复高斯随机变量的方差是其实部或虚部对应实高斯随机变量方差的两倍。也就是这里的n的方差是ρ(1−ρ)，所以n=c+dj的实部虚部c,d的方差都是ρ(1−ρ)2，所以c服从均值为0，方差为ρ(1−ρ)2的实高斯分布。这样就可以得到

f(c)=1πρ(1−ρ)exp[−(y−ρϕ[1−2b(4i)][2−(1−2b(4i+2))])2ρ(1−ρ)]

实部第一个bit的LLR

根据上面分布函数可以得到，实部第一个bit的LLR为l(4i)=lnP(b(4i)=0|y)P(b(4i)=1|y)=lnP(b(4i)=0,b(4i+2)=0|y)P(b(4i)=0,b(4i+2)=0|y)=exp[−(y−ρϕ)2ρ(1−ρ)]+exp[−(y−3ρϕ)2ρ(1−ρ)]exp[−(y+ρϕ)2ρ(1−ρ)]+exp[−(y+3ρϕ)2ρ(1−ρ)]

这个公式计算复杂，需要采用近似来简化。其中一个近似是log-sum-exp近似，也就是log(∑iexp(xi))≈max(xi)。这样可以得到l0≈max{−(y−ϕρ)2ρ(1−ρ),−(y−3ϕρ)2ρ(1−ρ)}−max{−(y+ϕρ)2ρ(1−ρ),−(y+3ϕρ)2ρ(1−ρ)}

这个近似又称为Max-Log-Map。

写成分段形式得到

l0≈{(y+3ϕρ)2ρ(1−ρ)−(y−ϕρ)2ρ(1−ρ), if y<−2ϕρ,(y+ϕρ)2ρ(1−ρ)−(y−ϕρ)2ρ(1−ρ), if −2ϕρ≤y<2ϕρ,(y+ϕρ)2ρ(1−ρ)−(y−3ϕρ)2ρ(1−ρ), if 2ϕρ≤y. ={8ϕ2ρ2+8ϕρyρ(1−ρ), if y<−2ϕρ,4ϕρyρ(1−ρ), if −2ϕρ≤y<2ϕρ,−8ϕ2ρ2+8ϕρyρ(1−ρ), if 2ϕρ≤y.

第二个近似得到（为什么这样近似还不确定，可以画图比较，会发现在正负0.5之间差距非常小，在正负1之间的差距可以接受），

l0≈4ϕρyρ(1−ρ)=4ϕy1−ρ

实部第二个bit的LLR

同样根据上面分布函数可以得到，实部第二个bit的LLR为l(4i+2)=lnP(b(4i+2)=0|y)P(b(4i+2)=1|y)=lnP(b(4i+2)=0,b(4i)=0|y)P(b(4i+2)=0,b(4i)=0|y)=exp[−(y−ρϕ)2ρ(1−ρ)]+exp[−(y+ρϕ)2ρ(1−ρ)]exp[−(y−3ρϕ)2ρ(1−ρ)]+exp[−(y+3ρϕ)2ρ(1−ρ)]

同样采用log-sum-exp近似，得到

l(4i+2)≈max{−(y−ϕρ)2ρ(1−ρ),−(y+ϕρ)2ρ(1−ρ)}−max{−(y−3ϕρ)2ρ(1−ρ),−(y+3ϕρ)2ρ(1−ρ)}

化简得到，

l(4i+2)≈{(y+3ϕρ)2ρ(1−ρ)−(y+ϕρ)2ρ(1−ρ), if y<0,(y−3ϕρ)2ρ(1−ρ)−(y−ϕρ)2ρ(1−ρ), if 0≤y=8ϕ2ρ2−4ϕρ|y|ρ(1−ρ)=8ϕ2ρ(1−ρ)−|l0|

虚部的两个bit的LLR计算类似。

64QAM

l0≈4ϕρyρ(1−ρ)=4ϕy1−ρ

l1≈16ϕ2ρ2−4ϕρ|y|ρ(1−ρ)=16ϕ2ρ(1−ρ)−|l0|
